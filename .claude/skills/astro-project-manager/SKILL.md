# astro-project-manager

适用于本项目：**星语 AstroWhisper** — 一个沉浸式星座占卜网站。

## 项目概况

- **前端**：Vite + React
- **后端**：Node / Express
- **AI 接入**：DeepSeek API（通过 OpenAI SDK 兼容模式）
- **当前已实现功能**：
  - 星座配对分析
  - 每日运势
  - 星卦问事（秒级起卦）
  - 情感起卦
  - AI 问题归类
  - AI 卦象解读
- **后续目标**（按优先级）：
  1. 后端算法层（起卦逻辑迁移到后端）
  2. SQLite 数据库
  3. 注册 / 登录
  4. 保存用户记录 / 历史卦象
  5. 部署上线

## 协作规范

处理本项目时请始终遵守以下规则：

### 沟通
- 始终使用**简体中文**回答和输出

### 修改流程
- 修改代码前先简要说明计划
- 用户不熟悉前端技术，尽量给出**完整文件内容**或**明确的替换方案**，避免只给片段

### 禁止操作
- 不要擅自修改 `.env`、`package-lock.json`、`node_modules`
- 不要删除文件
- 不要运行危险命令（`rm`、`del`、清空目录、重置 git 等）
- 不要随意安装依赖（如需安装，先说明理由征得同意）
